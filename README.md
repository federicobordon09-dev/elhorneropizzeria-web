# El Hornero Pizzería — Landing Page

**Aviso:** Este proyecto NO es la página oficial de El Hornero Pizzería. Es un proyecto personal / portafolio creado con fines educativos y de demostración. Busca replantear y mejorar la presencia digital de la pizzería, replicando sus datos reales (ubicación, contacto, carta) con una estética moderna, oscura y cálida.

---

## Sobre el proyecto

Landing page conceptual para **El Hornero Pizzería**, una pizzería artesanal de estilo napolitano ubicada en La Consulta, Mendoza. El sitio fue diseñado con una estética dark + glassmorphism: paleta negro, rojo ladrillo y crema, tipografía Display, animaciones sutiles con Motion y una experiencia optimizada para conversión por WhatsApp.

---

## Stack tecnológico

| Tecnología | Uso |
|---|---|
| Next.js 16 | Framework con App Router + Turbopack |
| TypeScript | Tipado estricto |
| Tailwind CSS v4 | Estilos utility-first con tokens personalizados |
| Motion (antes Framer Motion) | Animaciones scroll-reveal, carruseles y transiciones |
| Lucide React | Íconos SVG |
| Base UI (@base-ui/react) | Componente Button con render prop |
| next/font | Google Fonts: Inter + Playfair Display |

---

## Funcionalidades

- **Header** sticky con scroll-aware, menú hamburguesa mobile y navegación interna
- **Hero** full-viewport con resplandor de brasas animado, badge de horarios y CTA al menú
- **Nosotros** sección two-column con imagen + texto y 3 value cards con iconos (glass)
- **Por qué elegirnos** 6 benefit cards con iconos, hover elevado y glass
- **Galería** carrusel horizontal con auto-advance infinito (fotos reales del local)
- **Menú** cards de carta con lightbox: zoom progresivo (1x–6x), doble click, pellizco táctil y arrastre
- **Opiniones** ticker infinito horizontal con reseñas reales de Google Maps
- **Cómo funciona** 4 pasos conectados con línea punteada, grid responsive
- **Ubicación** mapa embebido de Google Maps + botón "Cómo llegar" con link corto
- **CTA final** card glass con resplandor y botón directo a WhatsApp
- **Footer** 3 columnas con logo, contacto (WhatsApp destacado como mini-botón), Instagram y horarios
- **Floating WhatsApp** botón flotante con animación pulse y parpadeo, aparece al hacer scroll
- **SEO** metadata completa, Open Graph, JSON-LD Schema.org (Restaurant), viewport y theme-color
- **Responsive** diseño mobile-first, touch targets ≥ 48px, animaciones adaptativas
