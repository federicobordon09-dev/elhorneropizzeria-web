Busca replantear y mejorar la presencia digital de la pizzería, replicando sus datos reales con una estética moderna, oscura y cálida.

---

## 📖 Sobre el proyecto

Landing page conceptual para **El Hornero Pizzería**, una pizzería artesanal de estilo napolitano ubicada en La Consulta, Mendoza. El sitio fue diseñado con una estética **dark + glassmorphism**: paleta negro, rojo ladrillo y crema, tipografía Display, animaciones sutiles con Motion y una experiencia optimizada para conversión por WhatsApp.

---

## ✨ Funcionalidades

### 🧭 Barra de navegación
- Sticky con scroll-aware
- Menú hamburguesa en mobile con navegación suave a secciones

### 🔥 Hero
- Full-viewport con **resplandor de brasas animado**
- Badge de horarios y CTA directo al menú

### 👨‍🍳 Sobre nosotros
- Two-column layout con imagen + texto institucional
- 3 value cards con glass effect

### ⭐ Por qué elegirnos
- 6 benefit cards con íconos, hover elevado y glassmorphism

### 🖼️ Nuestra especialidad
- Carrusel horizontal con **auto-advance infinito**
- Fotos reales del local

### 🍕 Menú
- **11 pizzas napolitanas** con nombre, descripción y precio
- **Lightbox** con zoom progresivo (1x–6x), doble click, pellizco táctil y arrastre

### 💬 Opiniones
- **Ticker infinito** horizontal con reseñas reales de Google Maps

### 📋 Cómo funciona
- 4 pasos conectados con línea punteada en grid responsive

### 📍 Ubicación
- Mapa embebido de Google Maps + botón "Cómo llegar"

### 🎯 CTA final
- Card glass con resplandor y botón directo a WhatsApp

### 📱 WhatsApp flotante
- Botón con animación pulse y parpadeo
- Aparece al hacer scroll

### 🔍 SEO
- Metadata completa con Open Graph y Twitter Cards
- **JSON-LD** con Schema.org Restaurant
- Viewport y theme-color optimizados

## 🛠 Stack tecnológico

| Tecnología | Uso |
|---|---|
| **Next.js 16** | Framework con App Router + Turbopack |
| **TypeScript** | Tipado estricto |
| **Tailwind CSS v4** | Estilos utility-first con tokens personalizados |
| **Motion** | Animaciones scroll-reveal, carruseles y transiciones |
| **Lucide React** | Íconos SVG |
| **Base UI** | Componente Button con render prop |
| **CVA + clsx + tailwind-merge** | Utilidades de clases condicionales |
| **next/font** | Google Fonts: Inter + Playfair Display |
